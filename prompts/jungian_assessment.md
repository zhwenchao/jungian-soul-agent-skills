# Jungian Assessment (English)

Based on Jungian analytical psychology, generate a structured psychological portrait of the user:

1. Conscious Ego: Introversion/Extraversion, Judging/Perceiving;
2. Core emotional response patterns;
3. Deep inner psychological needs;
4. Innate complexes and psychological blockages;
5. Archetypal foundation labels.

Output structured personality data readable by the Agent.

# Jungian Assessment (中文)

基于荣格分析心理学，生成用户结构化精神画像：

1.意识自我：内倾/外倾、判断/感知；

2.核心情绪反应模式；

3.深层内在心理需求；

4.原生情结与心理卡点；

5.人格原型基础标签。

输出可被Agent读取的结构化人格数据。
